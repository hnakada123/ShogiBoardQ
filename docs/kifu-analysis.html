棋譜解析

USIエンジンによる棋譜解析で形勢の推移を把握

評価値グラフ 候補手表示 解析範囲指定

解析の設定

棋譜解析ダイアログで解析条件を設定します

1 解析する棋譜を用意する

解析したい棋譜を用意します。対局終了後にそのまま解析することも、ファイルメニューから棋譜を読み込んで解析することもできます。

メイン画面：解析したい棋譜を表示した状態

2 棋譜解析ダイアログを開く

メニューの「対局」→「棋譜解析」から棋譜解析ダイアログを開きます。以下の項目を設定できます。

  • 解析に使用するUSIエンジンの選択
  • 解析範囲の指定（全手 / 指定範囲）
  • 1手あたりの思考時間の設定

棋譜解析ダイアログ：エンジン・解析範囲・思考時間を設定

解析結果

各手の評価値や候補手を確認できます

1 解析結果の表示

解析が完了すると、各手の評価値・候補手・読み筋が下部パネルに表示されます。棋譜の各手をクリックすると、その局面での解析結果を確認できます。

解析結果：各手の評価値・候補手・読み筋を表示

評価値グラフ

形勢の推移をグラフで視覚的に把握できます

1 評価値グラフの表示

評価値グラフタブでは、解析結果をもとに形勢の推移がグラフ表示されます。

評価値グラフ：形勢の推移を視覚的に表示

形勢判断の基準

評価値から形勢を5段階で判定します

評価値 形勢判断 表示例
0 ~ 100 互角 互角
101 ~ 300 やや有利 先手やや有利 / 後手やや有利
301 ~ 600 有利 先手有利 / 後手有利
601 ~ 1000 優勢 先手優勢 / 後手優勢
1001 以上 勝勢 先手勝勢 / 後手勝勢
詰み 勝ち 先手勝ち / 後手勝ち
  • 評価値は先手視点で、正の値は先手有利、負の値は後手有利を意味します
  • 形勢判断は評価値の絶対値をもとに判定されます
  • 詰みが検出された場合は、評価値の閾値に関わらず「先手勝ち」または「後手勝ち」と表示されます
