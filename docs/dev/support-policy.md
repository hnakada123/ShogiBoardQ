# サポートポリシー

## 対応対象

| 項目 | サポート方針 |
|---|---|
| Linux | x86_64 を公式サポート（AppImage 提供） |
| macOS | 12 以降を公式サポート（DMG 提供） |
| Windows | 10 / 11 (x86_64) を公式サポート（ZIP 提供） |
| Qt | 6.7 系をCIの基準バージョンとする |
| コンパイラ | GCC/Clang/MSVC の CI 対象構成を公式とする |

## リリース種別

- 安定版: タグ `v*` で配布するリリース。
- 開発版: `main` ブランチ上の成果物。互換性保証なし。

## 互換性方針

- 既存の棋譜フォーマット（KIF/KI2/CSA/JKF/USI/USEN/SFEN）の読み込み互換を維持する。
- 設定ファイル互換は `tst_settings_roundtrip` およびリリース前互換テストで確認する。
- 破壊的変更はリリースノートの「互換性注意事項」に必ず記載する。

## サポート期間

- 最新安定版を「フルサポート」対象とする。
- 直前安定版は重大不具合のみ対応対象とする。
- 2世代以上前の版は原則サポート外とする。

## サポート外

- 非公式ビルド（ローカル改造版）
- EOL済みOS
- 未サポートのQtメジャーバージョン組み合わせ
