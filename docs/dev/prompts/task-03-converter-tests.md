## CSA 変換に追加すべきテスト

### 正常系
1. **駒移動**: `+7776FU`（7七歩）→ 正しい SFEN 指し手への変換
2. **駒打ち**: `+0055KA`（5五角打ち）→ 駒打ち形式（`B*5e`）
3. **成り**: `+8822UM`（8八→2二馬）→ 成り指し手
4. **不成り**: `+8977KE`（桂不成り）→ 非成り指し手
5. **各種駒の移動**: 歩、香、桂、銀、金、角、飛、玉それぞれ
6. **対局開始 SFEN**: CSA 形式の初期局面 → 標準 SFEN
7. **駒落ち局面**: 平手以外の初期局面

### 異常系
8. **範囲外座標**: `+9977FU`（無効座標）
9. **不正な駒文字**: `+7776XX`
10. **空文字列**
11. **フォーマット不正**: `7776FU`（先頭の `+`/`-` なし）

## USI 変換に追加すべきテスト

### 正常系
1. **駒移動**: `7g7f` → 正しい SFEN 位置
2. **駒打ち**: `P*5e` → 歩の駒打ち
3. **成り**: `8h2b+` → 成り指し手
4. **不成り**: `8h2b` → 非成り指し手
5. **各種駒打ち**: `L*1a`, `N*3c`, `S*5e`, `G*4d`, `B*5e`, `R*5e`
6. **局面 SFEN + 指し手列**: `position startpos moves 7g7f 3c3d` 形式のパース

### 異常系
7. **範囲外座標**: `0a1b`（列0は無効）
8. **短すぎる文字列**: `7g`
9. **空文字列**
10. **不正な駒打ち文字**: `X*5e`
