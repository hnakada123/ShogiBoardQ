# Task 21: stale @todo コメントの一掃

## フェーズ

Phase 5（継続運用）- P1-4 対応

## 背景

`@todo remove コメントスタイルガイド適用済み` が 66 件残存しており、TODO がノイズ化して真に未対応の TODO を見分けにくい。

## 実施内容

1. 全対象を検索する：
   ```
   rg "@todo remove" src/
   ```

2. 各 `@todo remove コメントスタイルガイド適用済み` コメントを削除する
   - コメント行のみ削除（コードは変更しない）
   - コメントブロック全体が `@todo remove ...` のみの場合はブロックごと削除

3. **真の TODO**（まだ対応が必要なもの）がないか確認する：
   ```
   rg "TODO|FIXME|HACK|XXX" src/
   ```
   - `@todo remove` 以外の TODO は残す

4. ビルド確認: `cmake --build build`

## 完了条件

- `@todo remove コメントスタイルガイド適用済み` が 0 件
- 真に必要な TODO/FIXME は残っている
- ビルドが通る

## 注意事項

- コメント以外のコードは一切変更しない
- Doxygen コメント（`/** ... */`）内の場合、コメントブロックの構造を壊さないよう注意
